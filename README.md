#Common Task 1 Electron/Photon Classification

### Problem Description
Binary classification problem for 2-channel images of Electrons and Photons.

## Model Overview

### Optimizer
- **Optimizer:** ADAM
- **Training Epochs:** 20 epochs

### Results
- **Validation Accuracy:** 0.7366
- **Validation AUC:** 0.8058
